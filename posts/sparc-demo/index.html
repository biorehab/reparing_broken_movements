SPARC Interactive Demo

Companion simulation for the article on the SPARC measure of movement smoothness

SPARC (Spectral Arc Length) can be computed using the Monalysa Python library.

The sparc function takes the movement speed profile, along with four other parameters:

fs float
The sampling frequency of the data (Hz).
padlevel int, optional
Amount of zero padding applied to the speed profile before estimating the spectral arc length. Higher values give finer frequency resolution. Default: 4.
fc float, optional
Maximum cut-off frequency (Hz) for the spectral arc length calculation. Default: 10.0.
amp_th float, optional
Amplitude threshold used to determine the cut-off frequency up to which the spectral arc length is estimated. Expressed as a fraction of the DC magnitude. Default: 0.05.

Purpose of this simulation

See how the SPARC measure is computed, and how the three parameters — padlevel, max. cut-off frequency, and amplitude threshold — affect the result.

How to use it

Click Generate movement to produce a random four-submovement speed profile.

Understanding the different parameters

Playing around the with the parameters should make their roles clear, but here's a quick reference for what they all mean and how they affect the SPARC calculation:
padlevel
Zero-padding level is the amount of zero-padding applied to the time-domain signal before computing its Fourier transform to get a good approximation of the true continuous-time Fourier magnitude spectrum. Note that when you make the padlevel lower (0 ot 1) the DFT looses details and thus leads to underestimation of the arc length. Pad level of4 and above provides a good approximation, and thus the default value is set to 4. This is explained in more detail in our 2012 paper.
fcmax
Maximum Frequency cutoff is the maximum possible cut-off frequency upto which the arc length is computed. When the max. cut-off frequency is set lower, ignore important details of the Fourier spectrum can be ignored, thus underestimating the arc length.
amp_th
Amplitude threshold is the fraction of the DC value below which frequency components are considered negligible. Increasing the amplitude threshold will reduce the estimated cut-off frequency and thus lead to a more conservative estimate of the arc length.
