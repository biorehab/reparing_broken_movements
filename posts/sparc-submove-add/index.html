SPARC: Addition of Two Submovements

Companion simulation for the article on the SPARC measure of movement smoothness

Purpose of this simulation

See directly how adding a second submovement changes both the velocity profile and the Fourier spectrum, and watch the SPARC value respond in real time.

How to use it

Drag the two sliders. The interval slider moves the second submovement later in time. The amplitude slider controls how large it is relative to the first. Set amplitude to 0 for a perfectly smooth single-submovement baseline.

Output glossary

SPARC
Spectral Arc Length. Negative arc length of the normalised magnitude spectrum from 0 to fc. Less negative = smoother.
D
Movement duration, measured from the first to the last sample where speed exceeds 5 % of peak speed.
fc
Frequency cutoff: the highest frequency at which the normalised spectrum still exceeds 5 % of its DC value. This caps the arc-length integral.
